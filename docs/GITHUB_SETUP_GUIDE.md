# GitHub Setup Guide for Farjana Akther Koly Portfolio

This folder is ready for the repository:

```text
farjanaaktherkoly.github.io
```

## Part A — Upload Portfolio Website Files

1. Open GitHub.
2. Go to the repository: `farjanaaktherkoly.github.io`.
3. Click **Add file → Upload files**.
4. Upload these items directly into the repository root:

```text
index.html
styles.css
script.js
README.md
.nojekyll
assets/
```

Do not upload the outer folder itself. The `index.html` file must be visible in the main repository page.

5. Scroll down and click **Commit changes**.

## Part B — Enable GitHub Pages

1. Open the repository.
2. Go to **Settings**.
3. Click **Pages** from the left sidebar.
4. Under **Build and deployment**, select:

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

5. Click **Save**.
6. Wait 2–5 minutes.
7. Visit:

```text
https://farjanaaktherkoly.github.io/
```
